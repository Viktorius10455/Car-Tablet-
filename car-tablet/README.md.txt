# Car Tablet – FiveM

Een tablet menu waarmee je voertuigen kan spawnen via een mooie NUI layout.

## Command
- /cartablet

## Features
- Tablet menu
- Auto’s makkelijk toevoegen in server.lua
- NUI (HTML/CSS/JS)
- Standalone

## Waarom dit script?
Ik heb dit gemaakt om te leren:
- NUI menus maken
- client ↔ server communicatie
- voertuigen spawnen
- UI bouwen voor FiveM
